# HAINATIVE

**Defining Hybrid AI Native Systems**

HAINATIVE is the category and manifesto site within the HAIN ecosystem.  
It exists to define what Hybrid AI Native means, why it matters now, what principles shape it, and how it connects to architecture, protocol, and productization across local, edge, private, and cloud environments.

## What HAINATIVE does

- Defines the Hybrid AI Native concept
- Explains why this direction is emerging now
- Frames the core principles behind HAIN
- Connects the category to architecture, protocol, and productization
- Serves as the conceptual front door before deeper technical layers

## Initial site structure

- Home
- Why Now
- Model
- Principles
- Paths

## Position in the HAIN ecosystem

- **HAIN World** introduces the ecosystem
- **HAINATIVE** defines the category
- **HAINOA** explains the architecture
- **HAINPRO** maps and specifies the protocol layer
- **HAINIZE** operationalizes the system

## Working direction

HAINATIVE is not a product marketing site.  
It is a category site for articulating the Hybrid AI Native direction and helping visitors understand the broader HAIN system.
